Менің тәжірибем

Қолданған құралдар: ChatGPT, Gemini, DeepSeek

Жұмысты орындауда қандай құрал көмектесті?
Gemini маған API интеграциясын түсінуге көп көмектесті. DeepSeek ұзын құжаттарды талдауда тиімді болды.

ЖИ нені дұрыс немесе ыңғайсыз жасады?
Дұрысы: Негізгі код құрылымын тез ұсынды, қазақ тілінде жауап берді.
Ыңғайсызы: API кілтін орнату: Мен Google AI Studio арқылы API кілтін алғым келді бірақ ол істеу үшін 30 долор толтыру керек деген шарт тек карточка тіркегенсон шықты.

ЖИ 5 жылдан кейін әзірлеуші мамандығын қалай өзгертеді?
ЖИ күнделікті рутиналық тапсырмаларды автоматтандырады. Бірақ архитектуралық шешімдер, алгоритмдерді оңтайландыру, қауіпсіздік мәселелері әзірлеушілердің жауапкершілігінде қалады. көп ЖИ жаңа шығады және көбі тез жабылады ал тірі қалғандар чат гпт секілділер өздеріне ат жасап жаңа функциялар әкеледі.
