📝 Менің тәжірибем

Қолданған құралдар: ChatGPT, GitHub Copilot

Жұмысты орындауда қандай құрал көмектесті?
ChatGPT маған API интеграциясын түсінуге көп көмектесті. Асинхронды функциялар мен fetch сұраныстарын түсіндірді.

ЖИ нені дұрыс немесе ыңғайсыз жасады?
Дұрысы: Негізгі код құрылымын тез ұсынды. Ыңғайсызы: API кілтін орнату және қателерді өңдеу толық түсіндірілмеді. Мен оны өз қателіктерім арқылы түзеттім.

ЖИ 5 жылдан кейін әзірлеуші мамандығын қалай өзгертеді?
ЖИ күнделікті рутиналық тапсырмаларды (кодтың негізгі бөлігін жазу, тесттер жасау) автоматтандырады. Бірақ архитектуралық шешімдер, күрделі алгоритмдер, қауіпсіздік мәселелері әзірлеушілердің жауапкершілігінде қалады. ЖИ-мен жұмыс істей білу - жаңа міндетті дағдыға айналады.
