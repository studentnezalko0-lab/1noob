# 🤖 Әзірлеушіге арналған ЖИ-құралдары

Жасанды интеллект құралдарын насихаттайтын және Gemini API арқылы чат-ботпен өзара әрекеттесетін интерактивті веб-бет.

## 📋 Жоба туралы

Бұл веб-бет әзірлеушілерге арналған заманауи ЖИ-құралдарымен (**Grok**, **Gemini**, **DeepSeek**) таныстырады. Сонымен қатар, бетте Google Gemini API интеграцияланған чат-бот орналасқан, онда пайдаланушылар өз сұрақтарын қазақ тілінде қойып, ЖИ-дан жауап ала алады.

### Негізгі мүмкіндіктер:
- 🃏 **ЖИ-құралдарының карточкалары** - Grok, Gemini, DeepSeek туралы толық ақпарат
- 🌓 **Қараңғы/ашық тақырып** - бір батырмамен түс схемасын ауыстыру
- 💬 **Gemini чат-бот** - Google AI Studio API арқылы сұрақтарға жауап беру
- 📊 **Сұраныс санауыш** - API-ға неше рет сұрақ жіберілгенін көрсету
- ✨ **Анимациялар** - карточкалардың скролл кезінде пайда болу эффектісі
- 📱 **Адаптивті дизайн** - мобильді және десктоп құрылғыларға бейімделген
